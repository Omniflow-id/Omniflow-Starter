# AI Assistant - 上下文感知侧边栏

AI Assistant 是一个根据当前页面提供帮助的侧边栏 AI 助手。

## 什么是 AI Assistant？

AI Assistant 是一个从屏幕右下角滑出的聊天面板，当你点击带有 🤖 机器人图标的**蓝色**按钮时会显示出来。

与 AI Chat（完整页面）的区别：
- **AI Assistant**：快速侧边栏，具备上下文感知，不保存历史记录
- **AI Chat**：完整页面，基于对话，保存历史记录

## 如何使用

### 1. 打开 AI Assistant
点击屏幕右下角带有 🤖 图标的**蓝色**按钮。

### 2. 查看上下文
AI Assistant 会自动识别：
- **当前页面**：例如 "Users"、"Dashboard"、"Settings"
- **你的角色**：Admin、Manager 等
- **语言**：根据应用当前语言设置

### 3. 开始提问
输入你正在查看页面相关的问题：

**Users 页面示例：**
- "如何添加新用户？"
- "Admin 和 Manager 角色有什么区别？"
- "如何重置用户密码？"

**Dashboard 页面示例：**
- "解释显示的指标"
- "活跃用户和非活跃用户是什么意思？"

**Settings 页面示例：**
- "这个设置是做什么的？"
- "如何更改 X 配置？"

### 4. 实时响应
- AI 会流式输出回复（逐字符显示）
- 可滚动查看之前的内容
- 可通过展开按钮（↔️）放大面板

## 核心功能

### 上下文感知
AI 知道你当前在哪个页面，并据此调整回答。

### 语音输入（麦克风）
如果浏览器支持，可点击 🎤 麦克风图标进行语音输入。

### 文本转语音
点击 AI 消息上的 🔊 扬声器图标可朗读回答。

### 展开/收起
- **小尺寸**：400px（默认）
- **宽模式**：屏幕 50%（展开按钮）

### 历史导航
按浏览器“返回”按钮可以关闭面板（移动端友好）。

## 使用建议

### 提问要具体
❌ "这个怎么样？"
✅ "如何修改 Admin 角色权限？"

### 利用页面上下文
因为 AI 知道你当前页面，所以可以简短提问：
- "这个按钮有什么作用？"（同时指向屏幕上的按钮）
- "解释这个列"（同时指向表格）

### 多语言
AI 会根据应用语言响应（印尼语/英语）。

## 配置

AI Assistant 使用：
- [AI Analysis Settings](/admin/ai_analysis_settings) 来配置模型和参数
- 来自 `@knowledge/` 文件夹的知识库作为页面上下文

## 与其他 AI 功能的区别

| 功能 | 位置 | 保存历史 | 重点 |
|------|------|----------|------|
| **AI Assistant** | 侧边栏 | ❌ 否 | 当前页面帮助 |
| **AI Copilot** | 浮动 FAB | ❌ 否 | 屏幕分析 |
| **AI Chat** | 完整页面 | ✅ 是 | 深度对话 |

## 故障排查

| 问题 | 解决方案 |
|------|----------|
| 面板无法打开 | 刷新页面，检查 JavaScript 控制台 |
| 回答不相关 | 检查当前页面是否有可用知识库 |
| 语音输入无效 | 确保浏览器支持并已授予麦克风权限 |
| TTS 无效 | 确保浏览器支持 Web Speech API |
| 上下文错误 | 刷新页面以重置上下文识别 |

## 隐私说明

⚠️ AI Assistant 不会保存对话。每个会话都是全新的。

⚠️ 页面上下文在客户端读取，只有你的消息会发送到服务器。
