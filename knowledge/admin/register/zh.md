# 新账户注册

**注册**页面是用于在 Omniflow 系统中注册新账户的功能。此功能允许新用户自行创建账户，无需请求管理员。

## 适用人群

- **新员工** - 尚未在系统中拥有账户的用户
- **潜在用户** - 被邀请试用系统的用户
- **承包商/外包人员** - 由内部团队授权访问的人员

## 使用场景

1. **新员工入职** - HR 授予访问权限后
2. **用户试用** - 想要试用系统的用户
3. **邀请注册** - 使用管理员发送的邀请链接进行注册
4. **自助注册** - 如果管理员启用了此功能

## 注册步骤

### 1. 填写表单

在注册页面，请填写以下信息：
- **用户名**：登录名称（唯一，不能重复）
- **电子邮件**：有效的电子邮件地址（用于接收通知）
- **密码**：符合安全策略的密码
- **确认密码**：再次输入相同密码

### 2. 数据验证

系统将验证以下内容：
- 电子邮件格式正确
- 用户名未被使用
- 密码符合策略要求（长度、字符）
- 密码与确认密码一致

### 3. 验证（如果已启用）

如果管理员启用了电子邮件验证：
- 验证链接将发送到您的邮箱
- 点击链接以激活账户
- 激活后才能登录

### 4. 账户激活

成功后：
- 账户将立即激活（如果无需电子邮件验证）
- 或者点击验证链接后激活
- 即可登录

## 密码策略

密码必须满足以下要求：
- 最少 8 个字符
- 包含大写字母 (A-Z)
- 包含小写字母 (a-z)
- 包含数字 (0-9)
- 包含特殊字符 (!@#$%^&*)

## 错误信息及解决方案

| 错误信息 | 原因 | 解决方案 |
|---------|------|---------|
| 邮箱已被注册 | 邮箱已在系统中存在 | 使用其他邮箱或联系管理员 |
| 用户名已被使用 | 该名称已被其他用户使用 | 选择其他唯一用户名 |
| 密码不匹配 | 密码与确认密码不一致 | 确保两者完全相同 |
| 密码太弱 | 不符合密码策略要求 | 使用更复杂的组合 |
| 邮箱格式错误 | 邮箱格式无效 | 输入正确格式的邮箱 |

## 注册安全

### 邮箱验证

为防止虚假账户：
- 验证链接发送到邮箱
- 需要在规定时间内点击激活
- 如果未点击，将无法登录

### 垃圾信息防护

- 使用 CAPTCHA 验证（如果已启用）
- 限制注册频率以防止批量注册
- 对特定类型进行人工审核

## 如果无法注册

### 可能原因

1. **注册功能未启用** - 管理员尚未开启此功能
2. **仅限邀请注册** - 需要使用管理员提供的链接
3. **域名限制** - 仅允许特定的邮箱域名
4. **名额已满** - 用户数量已达上限

### 解决方案

- 联系管理员请求创建账户
- 如果已有邀请链接，请使用该链接
- 如正在维护中，请耐心等待

## 注册成功后

### 后续步骤

1. **登录** - 使用新账户登录系统
2. **完善资料** - 填写姓名、部门等信息
3. **修改密码** - 如果使用的是临时密码
4. **熟悉系统** - 如有培训请参加

### 如果使用邀请链接

- 账户已关联到特定团队/部门
- 无需输入额外信息
- 可直接访问相应角色对应的功能

## 故障排除

| 问题 | 解决方案 |
|------|---------|
| 无法访问注册页面 | 功能可能已禁用，请联系管理员 |
| 验证链接无法使用 | 检查邮箱，可能在垃圾邮件中，或请求重新发送 |
| 未收到验证链接 | 联系管理员进行手动激活 |
| 卡在验证流程 | 刷新页面，如果仍然卡住，请重新开始 |

## 与管理员账户的区别

| 方面 | 自助注册 | 管理员创建的账户 |
|------|---------|-----------------|
| 创建方式 | 用户自行创建 | 由管理员创建 |
| 密码 | 用户自行设定 | 可自动生成 |
| 验证 | 通常需要邮箱验证 | 直接激活 |
| 角色 | 默认角色 | 按需分配 |
