# Home Page ([/](/))

This is the landing page for omniflow-starter visitors.
Main features displayed:
1. **Hero Banner**: Displays the main value proposition.
2. **Our Services**: List of services offered (Web Dev, Mobile App, etc).
3. **Why Us**: Competitive advantages.
4. **Contact**: Address and phone number information.

## Visitor Guide
- Visitors can view service information.
- Visitors can contact us via WhatsApp button or Contact Form.
- Login is only required to access Client Area or Admin sections.

## Common FAQ
- **Is Omniflow free?** Yes, this starter version is open-source.
- **How do I login?** Click the Login button in the upper right corner.
