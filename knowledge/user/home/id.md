# Halaman Utama (Home)

Selamat datang di **Omniflow Starter** - Platform ERP modern untuk manajemen bisnis.

## Tentang Omniflow

Omniflow adalah sistem ERP (Enterprise Resource Planning) yang menyediakan solusi terintegrasi untuk:
- Manajemen Pengguna & Hak Akses
- Monitoring Sistem Real-time
- Manajemen Konten
- Automation & Workflow

## Fitur Utama

### 1. **Banner Hero**
Menampilkan value proposition utama Omniflow dengan call-to-action untuk mulai menggunakan sistem.

### 2. **Layanan Kami**
Daftar solusi yang ditawarkan:
- **Web Development**: Pembuatan aplikasi web custom
- **Mobile App Development**: Aplikasi Android & iOS
- **System Integration**: Integrasi sistem existing
- **Cloud Infrastructure**: Setup dan manajemen cloud
- **ERP Solutions**: Solusi ERP end-to-end

### 3. **Mengapa Memilih Kami**
Keunggulan kompetitif Omniflow:
- **Modern Stack**: Teknologi terkini (Node.js, React, PostgreSQL)
- **Scalable**: Arsitektur microservices yang scalable
- **Secure**: Security best practices & audit trails
- **Customizable**: Mudah dikustomisasi sesuai kebutuhan
- **Open Source**: Versi starter tersedia open-source

### 4. **Kontak**
Informasi kontak untuk konsultasi:

## Area Pengguna

### Client Area
Area khusus untuk klien dengan fitur:
- Dashboard project
- Monitoring progress
- Dokumentasi project
- Komunikasi dengan tim

### Admin Area
Area administrasi lengkap dengan:
- User Management
- Role & Permission System
- System Monitoring (Cache, Queue, Logs)
- AI-powered Features (Chat, Assistant, Copilot)

## Cara Menggunakan

### Untuk Pengunjung
1. Jelajahi informasi layanan di halaman ini
2. Hubungi kami via WhatsApp atau Form Kontak untuk konsultasi
3. Tim kami akan merespons dalam 1x24 jam

### Untuk Klien Terdaftar
1. Klik **Login** di pojok kanan atas
2. Masuk dengan akun yang telah diberikan
3. Akses Client Area untuk monitoring project

### Untuk Administrator
1. Akses `/admin` untuk login ke admin panel
2. Gunakan fitur manajemen lengkap
3. Monitor sistem via dashboard

## AI Features (Premium)

Sistem dilengkapi dengan fitur AI canggih:

### AI Chat
Interface percakapan AI untuk bantuan umum.

## FAQ Umum

**Q: Apakah Omniflow gratis?**
A: Versi starter (open-source) gratis untuk penggunaan. Versi enterprise dengan fitur lengkap tersedia dengan lisensi berbayar.

**Q: Bagaimana cara login?**
A: Klik tombol **Login** di pojok kanan atas halaman.

**Q: Apakah bisa request custom feature?**
A: Ya, silakan kontak tim kami untuk diskusi custom development.

**Q: Bagaimana keamanan data?**
A: Kami menggunakan enkripsi end-to-end, audit logs, dan security best practices.

**Q: Apakah ada dokumentasi lengkap?**
A: Ya, dokumentasi tersedia di area admin setelah login.
